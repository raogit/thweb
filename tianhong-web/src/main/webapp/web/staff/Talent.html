知识管理体系

天虹拥有完善的知识管理系统，包含E-learnning（老大师）系统与KMS系统。E-learnning系统可实现视频资料、微课、PPT资料的上传、存储与分享，实现全体员工的线上学习与考试。KMS系统可实现文档资料的知识沉淀与共享，可推送推荐至全体员工。
